# DevSteup
projet d'apprentissage de Git GitHub

##  Description
Ce dépôt regroupe mes notes et exercices sur **Git** et **GitHub** dans le cadre de mon apprentissage du développement web avec **Akieny Academy**.

##  Objectifs
- Comprendre Git et GitHub.
- Gérer les versions d'un projet.
- Utiliser les branches et les commits.
- Publier et collaborer sur GitHub.

##  Outils
- Git
- GitHub
- Visual Studio Code
- Git Bash

### Extensions VS Code recommandées
- GitLens
- Git Graph
- Prettier
- Live Server
- Auto Rename Tag

##  Commandes essentielles

```bash
git init
git status
git add .
git commit -m "Mon commit"
git branch
git checkout -b nom-branche
git merge nom-branche
git push origin main
git pull origin main
git clone URL
```

##  Utiliser un dépôt GitHub

1. Cloner un dépôt :
```bash
git clone URL
```

2. Ouvrir le projet.

3. Modifier les fichiers.

4. Enregistrer les changements :
```bash
git add .
git commit -m "Description"
git push origin main
```
